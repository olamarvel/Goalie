Welcome to Fun Fact Notifier

This application sends you fun facts periodically as notifications.

How to Use

  • Configure the settings on the Configuration Dashboard.
  • Set the notification interval, API key, and other preferences.
  • Click "Save Configuration" to start receiving notifications.
  • To start/stop/restart the app, use the provided buttons in the dashboard.
  • You can get your Api key from Api ninjas
  • Don't worry you won't be charged and the app is configurd to use the barest minimum request
